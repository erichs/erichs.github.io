Quick: what does this do?

 

  netstat -tn4 | awk '/ESTABLISHED/ {print $5}' | \
  sed 's/:.*$//' | sort | uniq

Shell functions to the rescue!

How about:

  unique_connected_hosts() {
    netstat -tn4 | awk '/ESTABLISHED/ {print $5}' | \
    sed 's/:.*$//' | sort | uniq
  }

Composition helps!

  unique_connected_hosts() {
    established_connections | strip_portnum | sort | uniq
  }

  established_connections() {
    netstat -tn4 | awk '/ESTABLISHED/ {print $5}'
  }

  strip_portnum() {
    sed 's/:.*$//'
  }
